## Getting Started

### Accessing the Merchant Portfolio

1. **Login** to the KYB Platform
2. **Navigate** to the main dashboard
3. **Click** on "Merchant Portfolio" in the navigation menu
4. You'll see the merchant portfolio overview page

### First Time Setup

- The system will display a welcome message with key features
- Review the portfolio types and risk levels
- Familiarize yourself with the search and filtering options

## Merchant Portfolio Overview

### Portfolio Types

The system categorizes merchants into four portfolio types:

- **Onboarded**: Fully verified and active merchants
- **Deactivated**: Merchants that have been deactivated
- **Prospective**: Potential merchants under evaluation
- **Pending**: Merchants awaiting verification

### Risk Levels

Each merchant is assigned a risk level:

- **High Risk**: Requires additional monitoring and compliance checks
- **Medium Risk**: Standard monitoring and periodic reviews
- **Low Risk**: Minimal monitoring requirements

### Portfolio Statistics

The overview displays:
- Total number of merchants
- Distribution by portfolio type
- Risk level breakdown
- Recent activity summary

## Search and Filtering

### Basic Search

1. **Enter search terms** in the search box
2. **Search fields include**:
   - Merchant name
   - Business address
   - Industry classification
   - Contact information

### Advanced Filtering

#### Portfolio Type Filter
- **Click** on portfolio type buttons to filter
- **Multiple selections** are supported
- **Clear filters** using the "Clear All" button

#### Risk Level Filter
- **Select risk levels** from the dropdown
- **Visual indicators** show risk level distribution
- **Combine with other filters** for precise results

#### Date Range Filter
- **Filter by creation date**
- **Filter by last activity date**
- **Use preset ranges** (Last 7 days, Last 30 days, etc.)

### Search Tips

- **Use partial matches**: "Acme" will find "Acme Corporation"
- **Case insensitive**: "acme" and "ACME" return the same results
- **Multiple terms**: "restaurant new york" finds restaurants in New York
- **Clear search**: Click the "X" button to clear search terms

## Troubleshooting

### Common Issues

#### Search Not Working
- **Check search terms** for typos
- **Clear filters** that might be restricting results
- **Try different search** terms or approaches
- **Refresh the page** if search appears stuck

#### Slow Performance
- **Reduce page size** for large merchant lists
- **Use filters** to narrow down results
- **Clear browser cache** and cookies
- **Check internet connection** speed

#### Session Issues
- **Refresh the page** to restore session
- **Log out and log back in** if session is lost
- **Clear browser data** if problems persist
- **Check browser compatibility** requirements

#### Data Not Updating
- **Refresh the page** to get latest data
- **Check for system notifications** about maintenance
- **Verify internet connection** is stable
- **Contact support** if data appears incorrect

### Browser Compatibility

#### Supported Browsers

## Glossary

**AML**: Anti-Money Laundering - regulations to prevent money laundering
**KYC**: Know Your Customer - verification of customer identity
**MCC**: Merchant Category Code - classification code for business types
**NAICS**: North American Industry Classification System
**Portfolio Type**: Category for merchant status (onboarded, deactivated, etc.)
**Risk Level**: Assessment of merchant risk (high, medium, low)
**SIC**: Standard Industrial Classification - business classification system
