## Getting Started (5 minutes)

### Step 1: Access the Platform

1. **Open your web browser** (Chrome recommended)
2. **Navigate** to the KYB Platform URL
3. **Log in** with your credentials
4. **Wait for the dashboard** to load

### Step 2: Navigate to Merchant Portfolio

1. **Look for "Merchant Portfolio"** in the main navigation menu
2. **Click** on Merchant Portfolio
3. **Wait for the portfolio page** to load
4. **Familiarize yourself** with the layout

### Step 3: Understand the Interface

The merchant portfolio page contains:
- **Search bar** at the top
- **Filter buttons** for portfolio types
- **Merchant list** in the center
- **Pagination controls** at the bottom
- **Statistics panel** on the right

## Key Features Overview (5 minutes)

### Feature 1: Advanced Search and Filtering

**What it does**: Helps you find specific merchants quickly

**How to use**:
1. **Use the search box** for merchant names, addresses, or industries
2. **Apply portfolio type filters** to focus on specific merchant groups
3. **Use risk level filters** to identify high-risk merchants
4. **Combine filters** for precise results

**Pro tip**: Save time by using filters before searching large datasets

### Feature 2: Merchant Detail Dashboard

**What it does**: Provides comprehensive information about a single merchant

**How to use**:
1. **Click on any merchant** to open their detail view
2. **Navigate between tabs** to see different information types
3. **Use the overview tab** for quick status checks
4. **Check the compliance tab** for regulatory status

**Pro tip**: Bookmark important merchants for quick access

### Feature 3: Bulk Operations

**What it does**: Allows you to perform actions on multiple merchants at once

**How to use**:
1. **Select merchants** using checkboxes
2. **Choose bulk actions** from the action menu
3. **Monitor progress** with the progress tracker
4. **Review results** when operations complete

**Pro tip**: Start with small batches (10-20 merchants) to learn the process

### Feature 4: Merchant Comparison

**What it does**: Lets you compare two merchants side-by-side

**How to use**:
1. **Select exactly two merchants** from the portfolio
2. **Click "Compare Merchants"** button
3. **Review the side-by-side** comparison
4. **Generate reports** if needed

**Pro tip**: Use comparison for risk assessment and decision-making

### Feature 5: Session Management

**What it does**: Keeps track of your current merchant context

**How to use**:
1. **Notice the current merchant** indicator in the navigation
2. **Switch merchants** using the merchant picker
3. **Use breadcrumb navigation** to move between merchants
4. **Session persists** across page navigation

**Pro tip**: Use session management to maintain context while working

## Common First-Time User Questions

### Q: How many merchants can I manage?
**A**: The platform supports thousands of merchants in your portfolio. Start with your current merchants and add more as needed.

### Q: Can I customize the interface?
**A**: Yes, you can adjust page sizes, save search preferences, and customize your view settings.

### Q: Is my data secure?
**A**: Yes, the platform uses enterprise-grade security with encryption, audit trails, and compliance with industry standards.

### Q: Can I export merchant data?
**A**: Yes, you can export merchant data in various formats (CSV, Excel, PDF) for reporting and analysis.

### Q: What if I make a mistake?
**A**: Most actions can be undone, and the system maintains audit trails. Contact support if you need help with corrections.

### Q: How do I get help?
**A**: Use the in-app help system, live chat, email support, or phone support. All contact methods are available 24/7.
